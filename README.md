# 💸 SplitMate – No More “Who Owes Whom?” Chaos

SplitMate is a lightweight group expense tracker designed to make life easier for broke uni squads, travel groups, and friend hangouts. No signup, no database—just open it, enter your group expenses, and see who needs to pay whom.

## ✨ Features

- ✅ Create a group with custom members  
- ✅ Add multiple paid situations (bus, food, gifts...)  
- ✅ Split bills equally or by custom amounts  
- ✅ See a final summary of who should give/receive money  
- ✅ Copy summary to clipboard (perfect for WhatsApp 😉)  
- ✅ Reset all data in one click  

---

## 🛠 Tech Stack

- **Framework:** Next.js 13+ with App Router  
- **Language:** TypeScript  
- **Styling:** Tailwind CSS  
- **State:** React Hooks (`useState`, `useReducer`)  
- **Storage:** No database (for MVP – all in memory)
